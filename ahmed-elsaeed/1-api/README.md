# Simple Static Project

**Project**: A minimal static site containing `index.html`, `main.js`, and `style.css`.

**Files**:
- `index.html`: Entry page
- `main.js`: JavaScript for the page
- `style.css`: Styles

**Getting Started**:
- **Open:** Double-click `index.html` or open it in your browser.



**Development**:
- Edit `index.html`, `main.js`, or `style.css` in your editor and refresh the browser.
